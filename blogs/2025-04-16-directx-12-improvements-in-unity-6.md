Greetings from the Unity Graphics team!In this post, we will cover the latest improvements to the functionality and performance of the DirectX 12 (DX12) graphics backend. As of Unity 6.1, DirectX 12 is now set as the default graphics API for new projects.DX12 provides a modern, lower level and thinner abstraction for graphics acceleration.
