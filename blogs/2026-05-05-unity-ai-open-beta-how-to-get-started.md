Unity AI is a suite of tools built into the Unity Editor for developers using Unity 6 or later. Unity AI includes the AI Assistant (Ask, Plan, and Agent modes), the MCP Server for connecting IDE agents to your project, AI Gateway for routing existing model subscriptions (Claude, GPT-4o, Gemini) through the Editor, and asset Generators for sprites, textures, and scene layouts.
