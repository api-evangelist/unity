The Unity MCP Server implements the Model Context Protocol to give IDE-based AI agents (Claude Code, Cursor, Windsurf, VS Code Copilot) live access to a running Unity project — including scene hierarchy, GameObjects, component values, console output, and build settings. Setup requires enabling the Unity bridge in Project Settings, configuring your client via the Integrations panel or a manual relay path, and approving the connection.
